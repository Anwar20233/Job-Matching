# Smart Saudi Job-Matching Chatbot

**Smart Saudi Job-Matching Chatbot** هو نظام تفاعلي ذكي لمطابقة الوظائف في السعودية، مصمم لتقديم توصيات دقيقة وملائمة للمستخدمين بناءً على بيانات سوق العمل المحلية.

---

## 💡 نظرة عامة
يهدف المشروع إلى تحسين تجربة البحث عن الوظائف عبر توفير **دردشة تفاعلية ذكية** تعتمد على تقنيات حديثة لمعالجة اللغة العربية وفهم سياق المستخدم.  
المشروع يجمع بين:

- **مجموعة بيانات Jadarat** للوظائف السعودية.
- **تنظيف النصوص العربية** لضمان جودة المعالجة.
- **نموذج Transformer LM** مبني من الصفر لتوليد ردود ذكية.
- **نظام RAG (Retrieval-Augmented Generation)** باستخدام **TF-IDF + FAISS** لاسترجاع معلومات دقيقة.
- **واجهة تفاعلية عبر Gradio** لتجربة سلسة وودية للمستخدم.

---

## 🛠️ مكونات المشروع

### 1. معالجة البيانات
- تنظيف النصوص العربية (حذف الرموز، تصحيح التشكيل، إزالة الكلمات غير المهمة).
- تحويل البيانات إلى صيغة مناسبة للنماذج العصبية.

### 2. نموذج اللغة Transformer
- نموذج من الصفر قادر على فهم السياق العربي.
- توليد ردود ذكية ومتسقة مع استعلامات المستخدم.

### 3. نظام RAG
- **TF-IDF** لاستخراج الكلمات المفتاحية الأكثر أهمية.
- **FAISS** للبحث السريع عبر قاعدة البيانات.
- دمج النتائج مع نموذج اللغة لتوليد إجابات دقيقة وموثوقة.

### 4. واجهة المستخدم
- **Gradio Interface** للتفاعل السلس.
- إمكانية إدخال استفسارات المستخدم واستلام توصيات الوظائف في الوقت الحقيقي.
